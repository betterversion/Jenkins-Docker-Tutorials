In the jobs folder you will find a folder per example in this course, open up the relevant example by job name that you see when watching the course. Then there's a config.xml file, you can load this job right into Jenkins. See the end of module 2 - "Challenge And Importing Job config.xml Files" for instructions.

Also, there are links for the course here, including some additional instructions for following along with Windows:

https://git.io/vKSVZ

Required software
- java 8
- git

All required software must be in the PATH of the user you run jenkins as.

Windows users use the "jobs-windows" folder.

Mac/linux/unix users use the "jobs-nix" folder.
